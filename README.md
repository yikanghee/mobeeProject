영화 웹 어플리케이션

# 목표
* 영화 사이트를 직접 구현하여 Bank-End System을 이해한다.
* React.js를 활용하여 Front-End를 직접 구현하여 이해한다.
* JPA를 사용하여 객체지향 프로그래밍을 이해한다.

# 사용기술
* Spring Boot, Java 8, Gradle, Mysql, JPA, Redis, React, Redux, JWT, EC2, S3
